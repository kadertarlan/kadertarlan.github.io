Appium Kurulumu

Appium kurulumu kullanımı. İlk olarak Eclipse kurulu değil ise Eclipse IDE‘yi indiriyoruz kullandığınız platforma göre linkten bulabilirsiniz. http://www.eclipse.org/downloads/ Ayrıca JDK indirmeniz gerekiyor. Bunun için de aşağıdaki linkten, işletim sisteminizi seçip anlaşmayı onayladıktan sonra indirebilirsiniz. http://www.oracle.com/technetwork/java/javase/downloads/jdk8- downloads-2133151.html JDKʼ yı indirip kurun. Yükleme bitince Eclipseʼyi açın. Android için SDK indireceğiz. Bunun için aşağıdaki linkten indirme işlemini yapıyoruz. http://www.gezginler.net/indir/android-sdk.html İndirdikten sonra çift tıklayıp kuralım. Eclipse dosyasının içine kurabilirsiniz. Yükleme bitince SDK Manager programı açılacaktır. Burada yüklemek istediğimiz Android sürümlerini seçerek kurulumunu sağlayabilirsiniz. Şimdi Eclipseʼyi açın ve yukarıdaki menüden Help-Install New Software seçeneğine gelin. Location kısmına https://dl-ssl.google.com/android/eclipse/ adresini girin. ADT (Android Development Tools) için gelen verileri seçerek indirilmesini sağlayın. Son olarak Eclipse içerisinde Window-Preferences ve soldaki Android kısmına tıklıyoruz. Buradaki SDK Location bölümüne Android SDKʼyı nereye kurduysak onu işaretliyoruz. Kurulum işlemlerimiz tamam. Android Tools indirirken Android DDMSʼde inmiş oldu. Bu bizim Android testlerimiz için mobil cihazın ekran görüntüsü almamızı sağlayacak. Şimdi Appium kuralım.https://bitbucket.org/appium/appium.app/downloads/ buradan Appium versiyonunu projenize uygun şekilde seçmelisiniz. Ben Android 5 ve 6 sürümlerinde ayrıca ios 9 sürümünde test yapabilmek için 1.4.13 indirdim. Mac bilgisayarlar için kurulumu oldukça basit indirdiğiniz dosyanın üzerine gelerek kurulumu gerçekleştiriniz. appium.jpeg Burada localde bir android telefonu bilgisayarınıza bağlayıp resimde görüldüğü gibi Android iconuna tıklayıp Launch ettiğinizde status 200 alcaksınız başarılı bağlanma sonunda. appium2 Kurulumlar tamamlandıktan sonra Could not find adb. Please set the ANDROID_HOME environment variable with the Android SDK root directory path. Gib Bir hatamız local bunun için vi ~/.bash_profile girip export ANDROID_HOME=/Applications/Android export PATH=$ANDROID_HOME/platform-tools:$PATH export PATH=$ANDROID_HOME/tools:$PATH Bunları yazmak gerekiyor dosyaya.

https://developer.apple.com/download/more/ Buradan xcode 7.3.1

https://nodejs.org/en/download/ İnstall nodejs node -v diyoruz v6.9.2

Sonra path nodejs tanımla export PATH=/usr/local/bin/node:$PATH

. http://brew.sh/index_tr.html sitesinde linke(/usr/bin/ruby -e “$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)”) ulas kopyala terminale yapistir ilk gelen soruda Brew update Brew doctor

sudo npm install -g appium Ve https://bitbucket.org/appium/appium.app/downloads/ buradan appium 1.4.13 kuralım

Şimdi java jdk indirelim http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html?ssSourceSiteId=otnes* Buradan Java SE Development Kit 8 indirdik. Kurulum tamamlayalım

Şimdi java -version diyerek kurulum kontrol edelim. echo export “JAVA_HOME=\$(/usr/libexec/java_home)” >> ~/.bash_profile ile java pays tanımlayalım bash_profile

http://www.eclipse.org/downloads/download.php?file=/technology/epp/downloads/release/mars/2/eclipse-jee-mars-2-macosx-cocoa-x86_64.tar.gz Buradan eclipse indirrelim.

https://developer.android.com/studio/index.html#downloads Android sdk indirelim.

ADT ile android gerekliliklerini indirelim eclipse içinde (help > install new software) ADT Plugin - https://dl-ssl.google.com/android/eclipse

Eclipse için adnroid gerekliliklerini indiriyorz.

Konsolda bash profile içine: export PATH=“/usr/local/bin:$PATH” export PATH=/usr/local/lib/node_modules/grunt/bin:$PATH export PATH=/usr/local/bin/node:$PATH export JAVA_HOME=$(/usr/libexec/java_home) export ANDROID_HOME=“/Users/kader.tarlan/Downloads/tools/” export PATH=“/Users/kader.tarlan/Downloads/tools/tools:$PATH” export PATH=“/Users/kader.tarlan/Downloads/tools/platform-tools:$PATH” export PATH=“/Users/kader.tarlan/Downloads/tools/build-tools:$PATH”
